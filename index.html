Introducing the Ultradrone

Your All-in One Modular Drone Platform

About

Ultradrone is a Modular Drone that has built in componets that allows it to be useful in a variety of stituations.

What we've done is build a drone from scrath that can support integrating modular componenets underneath the central platoform of the drone.

How It Works

One application of the Ultradrone is in the use of a Security Drone.

Ultradrone's modular capability allows us to add componenets and integrate into it's existing platform.

For example you would connect your camera to the onboard ardunio and remotely run your code
